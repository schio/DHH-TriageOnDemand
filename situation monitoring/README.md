DHH-TriageOnDemand 재난상황 모니터링
-----------

ZEPPELIN을 통한 정보 제공

1. 대시보드 - 전체 환자 수, 트리아지 구분 별 환자 수, 전체 환자 리스트, 외부물자 리스트 등 제공
2. 환자 상세 검색 - 이름으로 환자 검색, 사진으로 환자 검색, 환자 상세 정보 제공
3. 각 재난 종료 후 결과보고 - 예방 사망률, ROC 커브 등
